# TXO 周選擇權賣方策略知識庫

## 策略概述

**Bull Put Spread（牛市賣權價差）**，每週操作台指選擇權（TXO）賣方，以收取時間價值為主要獲利來源。

- **賣出 Put**：週開盤 × 98%（價外 2%），Delta ≈ -0.20
- **買入 Put**：週開盤 × 96.5%（價外 3.5%，保護腳）
- **到期日**：每週三（台灣周選擇權到期日）
- **合約乘數**：50 元/點

---

## 5條執行規則（基於1年回測，2025-03 ～ 2026-03）

### 規則一（必要）：TWII 站上 MA20 才進場

```python
entry_allowed = twii_current > ma20
```

效果：過濾 2025/4 黑天鵝（-13.7%）、11月閃崩（-4.62%）等 4 個虧損週。
未過濾總損益 +NT$3,316 → 加此規則後 +NT$40,658。

### 規則二（必要）：前週漲幅 > 4% 則跳過

```python
entry_allowed = prev_week_return < 4.0
```

效果：避開 2026/2/26（-7.41%，虧損 NT$23,364）。該週前週漲幅剛好 +5.20%。
加此規則後 MaxDD 從 -23,364 縮小至 -12,032（減少 57%）。

### 規則三：停損設定

- 中途停損條件：大盤跌破賣出履約價 1%（例：賣 33,000 → 跌破 32,670 立即平倉）
- 金額停損：浮虧達收到權利金 2 倍時平倉

### 規則四：提前獲利了結

- 賣出 Put 剩餘價值縮小至 20% 以下時，買回平倉（不等到到期）
- 目標：鎖定 80%+ 獲利，釋放保證金

### 規則五：資金控管

- 每次操作保證金不超過帳戶 30%
- 月最大虧損上限：帳戶 5%

## 進場執行步驟

```
1. 確認 TWII > MA20（用 fetcher_factory.py 取得 1個月日線）
2. 確認前週漲幅 < 4%（用週線 open/close 計算）
3. 計算賣出履約價 = 本週開盤 × 0.98（取整百）
4. 計算買入履約價 = 本週開盤 × 0.965（取整百）
5. 用 Black-Scholes 估算淨收點數（IV 假設 20%，DTE=5）
6. 確認淨收 ≥ 開盤價 × 0.0012（即約每口 NT$2,000 以上）才值得進場
```

## Black-Scholes 估價函數

```python
import math

def norm_cdf(x):
    return 0.5 * (1 + math.erf(x / math.sqrt(2)))

def bs_put(S, K, T, sigma=0.20, r=0.015):
    """
    S: 現貨價格 (TWII)
    K: 履約價
    T: 到期天數 / 365
    sigma: 隱含波動率（預設 20%）
    r: 無風險利率（預設 1.5%）
    """
    if T <= 0 or K <= 0: return 0
    d1 = (math.log(S/K) + (r + sigma**2/2)*T) / (sigma*math.sqrt(T))
    d2 = d1 - sigma*math.sqrt(T)
    return max(0, K*math.exp(-r*T)*norm_cdf(-d2) - S*norm_cdf(-d1))
```

## 台灣市場特殊注意事項

- **到期日**：每週三（若逢假日順延）
- **最後結算**：以到期日開盤競價加權平均（非收盤價）
- **流動性**：近月近價較佳，深度價外流動性差 → 使用限價單
- **交易稅**：買賣各 0.1‰
- **保證金**：Bull Put Spread 保證金 = 最大損失（固定）
- **週選代碼**：TXO + 履約價 + W + 週別（如 TXO33000W2）

## MA20 計算方式

```bash
# 取近 30 個交易日的收盤價計算 MA20
uv run python fetcher_factory.py price_history ^TWII \
  --market TW --start $(date -v-45d +%Y-%m-%d) --end $(date +%Y-%m-%d)
```

或用 yfinance 直接計算：
```python
import yfinance as yf
hist = yf.Ticker('^TWII').history(period='2mo')
ma20 = hist['Close'].rolling(20).mean().iloc[-1]
```
